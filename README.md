# Diabetes Prediction System

## Overview
This project predicts whether a person is diabetic or non-diabetic using a Random Forest Classifier.

## Features
- Data preprocessing
- Feature scaling
- Random Forest classification
- Model evaluation
- User input prediction
- Risk percentage prediction

## Technologies Used
- Python
- Pandas
- NumPy
- Scikit-learn

## Dataset
Diabetes Health Indicators Dataset
